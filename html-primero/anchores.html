HIPERVÍNCULOS

Se pueden utilizar hipervínculos de distintas maneras. Sin embargo, es necesario incluirle siempre el atributo "href".
A continuación se muestran algunos ejemplos del uso del hipervínculo. 

                
<a href="mailto:potato_riko@outlook.com">Enviarle un correo</a>
<a href="tel:+51998998998">Realizar una llamada</a>
<a href="https://wa.me/15551234567" target="_blank">Realizar un whatsapazo!</a>

Lo mostrado en el bloque de código anterior se traduce a lo siguiente:

  • Enviarle un correo
  • Realizar una llamada
  • Realizar un whatsapazo!


Un elemento <a> también puede utilizarse para "envolver" ciertos elementos, es decir, se pueden incluir elementos diferentes a textos dentro de un hipervínculo.

HOLA, SOY UN DIV XD

Si se le realiza un click al primer o al segundo elemento, se podrá notar que aparecerá un diálogo de descarga automática. Esto es posible gracias al atributo download el cual permite una descarga instantánea a partir de un click. Eso sí, solo se podrán descargar archivos que sean locales o pertenezcan al dominio de la aplicación. Si el archivo pertenece a otro dominio, se abrirá una nueva pestaña.
